Document Summary App


Automated Document Actions and Interactive Q/A

This is a simple, self-contained application that accepts a document or URL,
summarizes the content using Google’s Gemini model, answers user questions,
and simplifies document sections for easier understanding.





Features :

- Upload PDF or TXT files

- Enter a URL to fetch web content

- Generate summaries in short, detailed, or bullet point styles

- Ask questions and get relevant answers based strictly on the document

- Simplify sections or topics for a younger audience

- Clear, easy-to-use web interface with logging of actions and errors





How to Run :

- Unzip the project folder

- Double-click run_app.bat to start the app

- Your default browser will open the interface

- Upload a file or enter a URL

- Generate summary, ask questions, or simplify content using the buttons





File Structure :

doc_summary_app/
├── app.py              # Main app script
├── run_app.bat         # Shortcut to launch the app
├── app_log.txt         # Log file generated during runtime
└── README.md           # This file
